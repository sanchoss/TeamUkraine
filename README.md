### 📁 Структура папок
- **roster.md** — Склад команди, категорії, доступність
- **events/** — Серії гонок: рік → серія → подія
- **results/** — Результати, скріншоти, дані ZwiftPower
- **discord/** — Шаблони оголошень для Discord

### 📝 Як додати нову подію
1. Скопіювати `events/_template_event.md`
2. Помістити в `events/YYYY/НАЗВА_СЕРІЇ/`
3. Заповнити деталі (дата, маршрут, склад, результати)

### 📁 Folder Layout
- **roster.md** — Current team members, categories, availability
- **events/** — Race series organized by year → series → event
- **results/** — Finish data, screenshots, ZwiftPower exports
- **discord/** — Announcement templates, pinned messages

### 📝 How to Add a New Event
1. Copy `events/_template_event.md`
2. Place it in `events/YYYY/SERIES_NAME/`
3. Fill in details (date, route, lineup, results)
